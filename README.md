VibeCheck is a high-energy, AI-integrated e-commerce platform built during GDG Kolachi's Build with AI Workshop 03, breaking the "minimalist" norm with a Neobrutalist, Gen-Z design language.

| Layer | Technology |
|---|---|
| Frontend | HTML, CSS, JavaScript, TypeScript |
| Design | Neobrutalist UI, Bento-grid layouts |
| Deployment | Vercel |

### 📄 AI Resume & ATS Evaluator

An AI-powered Applicant Tracking System (ATS) Evaluator that helps candidates optimize resumes for modern hiring workflows, using LLMs for semantic matching between a candidate's experience and job descriptions.

| Layer | Technology |
|---|---|
| Frontend | HTML, CSS, JavaScript, TypeScript |
| AI/Logic | LLM-based Semantic Matching |
| Deployment | Netlify |

🔗 **[Live Demo](https://atsscoreevaluator.netlify.app/)** &nbsp;|&nbsp; 💻 **[Code](https://github.com/areefasamar/Web-Development/tree/main/ATS-Score-Evaluator)**
